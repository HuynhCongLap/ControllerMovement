Pour compiler sous Linux
========================

1. Installer CodeBlocks et les libs glfw3 et glew :

sudo apt install codeblocks
sudo apt install libglfw3-dev libglew-dev

2. Ouvrir et compiler les projets :
- Box2D\CB_projectFiles\Box2D.cbp
- Box2D\CB_projectFiles\IMGUI.cbp
- TP_CONTROL_BIPED_Linux.cbp
